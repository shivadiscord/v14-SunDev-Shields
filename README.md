# v14 Guard Bot

Discord sunucunuzun güvenliğini sağlayan, v14 altyapısına sahip güçlü koruma botu.

---

## 🚀 Özellikler
- Rol & kanal koruması
- Yetki değişim engelleme
- Spam / raid koruması
- Log sistemi
